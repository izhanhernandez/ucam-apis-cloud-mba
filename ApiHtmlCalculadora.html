Servicio Calculadora API:

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

Funciones:

- suma:

Realiza una suma de las variables que se introduzcan como parametros.

Parametro Descripcion Obligatorio Valor por defecto
variable1
 Primera variable para sumar. SI N/A
variable2
 Segunda variable para sumar. SI N/A

Peticion ejemplo:

http://localhost:5000/suma?variable1=5&variable2=3

 

- resta:

Realiza una resta de las variables que se introduzcan como parametros.

Parametro Descripcion Obligatorio Valor por defecto
variable1
 Primera variable para restar. SI N/A
variable2
 Segunda variable para restar. SI N/A

Peticion ejemplo:

http://localhost:5000/resta?variable1=5&variable2=3
